# cse-144-final-project
Final Project: Kaggle Transfer Learning Challenge for CSE 144: Applied Deep Learning @ UCSC (Spring 2026)

**1. Training/validation accuracy (and loss) and key curves.**

The final configuration reaches **78.6% validation accuracy (169/215)** on the
held-out split. The train-vs-validation loss curve below shows both losses
dropping quickly in the first ~5 epochs and then flattening, with validation loss
tracking (and staying slightly below, thanks to label smoothing + dropout +
MixUp/CutMix) training loss — indicating regularization is keeping overfitting in
check.

![Train vs Validation Loss](loss_curve.png)

**2. Kaggle public leaderboard score (and submission settings).**

**76.36%** on the Kaggle public leaderboard. Submission was generated by running
the final saved model (`model.pth`) over all test images with the deterministic
384×384 eval transform and writing `submission.csv` via `predict.py`. (The public
leaderboard reflects ~10% of the test set and is used only to confirm submission
format, per the instructions.)
